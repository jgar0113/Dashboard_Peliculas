# Dashboard de Películas

Dashboard interactivo desarrollado con Streamlit y Firebase.

## Funcionalidades
- Visualización del listado de películas
- Búsqueda por título (coincidencia que contenga, sin distinguir mayúsculas)
- Filtro por director
- Registro de nuevas películas
- Datos almacenados en Firestore

## Tecnologías
- Python
- Streamlit
- Firebase Firestore

## Consideraciones

Por seguridad, el archivo de credenciales 'serviceAccountKey.json' **no se incluye en el repositorio**
